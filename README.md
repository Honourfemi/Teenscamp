# Teens Camp 4th Edition — Website

Built with Next.js, Tailwind CSS, Supabase, Paystack, and Resend.

## Setup checklist

1. Run `supabase-schema.sql` in your Supabase project's SQL Editor.
2. Copy `.env.example` and fill in real values as Environment Variables in Vercel.
3. Edit `lib/campConfig.ts` with your real camp date, fee, and contact info.
4. Deploy on Vercel (connected to this GitHub repo).

See the full step-by-step guide from your assistant for exact instructions.
